1. 핵심 기술 및 아키텍처 학습

React

  • •컴포넌트 구조 및 상태 관리 실습
  • •JSX와 이벤트 핸들링 이해
  • •React Router를 활용한 라우팅 설계, 성능 최적화 기법을 학습

React는 현대 프론트엔드 개발의 핵심 라이브러리입니다. IT를 처음 접하는 분들을 위해 HTML/CSS 기초부터 시작하며, 가상 DOM의 개념을 이해하고, 컴포넌트 기반으로 재사용 가능한 UI를 설계합니다. Hooks를 활용한 상태 관리, 라우팅 구성, 성능 최적화까지 실무에서 요구하는 모든 React 기술을 학습합니다.

Node.js

  • •이벤트 루프와 논블록킹 I/O 동작 이해
  • •Express로 REST API 구성
  • •비동기 패턴(Promise, async/await)과 실시간 통신(Socket.io)

Node.js는 JavaScript로 백엔드를 개발할 수 있게 해주는 런타임 환경입니다. TypeScript와 ES6+ 최신 문법을 먼저 익히고, V8 엔진의 동작 원리와 이벤트 루프를 이해합니다. Express로 RESTful API를 만들고, 회원가입/로그인, OAuth 소셜 로그인, 파일 업로드, 실시간 채팅 등 실무에서 자주 사용하는 기능을 직접 구현합니다. MongoDB와 NestJS까지 학습하여 현대적 Node.js 백엔드 개발 역량을 갖춥니다. 터미널 명령어와 Git 사용법도 함께 훈련합니다.

Java Spring

  • •Spring의 의존성 주입과 Boot 기반 서버 구성
  • •JPA와 트랜잭션 관리
  • •MVC 아키텍처 및 테스트 전략

Java Spring은 국내 대기업과 스타트업에서 가장 많이 사용하는 백엔드 프레임워크입니다. 의존성 주입(DI)과 Spring Boot의 기본 구조를 배우고, JPA로 데이터베이스를 효율적으로 관리하는 방법을 익힙니다. 블로그 프로젝트를 만들면서 Spring Security로 보안을 설정하고, JWT와 OAuth2를 활용한 인증 시스템을 구현합니다. MySQL 데이터베이스와 연동하여 실전 수준의 백엔드 서비스를 완성합니다.

동기/비동기, 블록킹/논블록킹

  • •동기/비동기 처리와 블록킹/논블록킹 모델의 차이를 이해
  • •비동기 API 설계 및 병렬 처리
  • •이벤트 기반 아키텍처 구현 방법을 학습

웹 개발에서 가장 중요한 개념 중 하나인 동기/비동기와 블록킹/논블록킹의 차이를 명확히 이해합니다. Node.js의 V8 엔진 설명과 함께 진행되며, 서버가 여러 요청을 동시에 처리하는 원리를 배웁니다. 비동기 API를 설계하고, 병렬 처리와 이벤트 기반 아키텍처를 통해 고성능 서버를 구축하는 방법을 학습합니다.

생성형 AI를 활용한 풀스택 개발

  • •백엔드/프론트엔드 협업 사례 분석
  • •AI 기반 코드 보조 시스템 구현
  • •사용자 피드백 데이터를 활용한 개선 전략

최신 AI 기술을 풀스택 개발에 적용하는 방법을 배웁니다. ChatGPT API를 백엔드와 프론트엔드에 연동하여 챗봇, 자동 응답 시스템, 콘텐츠 생성 기능 등을 구현합니다. AI 기반 코드 자동 생성 도구를 활용하여 개발 효율을 높이고, 사용자 피드백을 분석하여 서비스를 지속적으로 개선하는 전략을 학습합니다.

SSR과 MSA

  • •서버사이드 렌더링(SSR) 기초
  • •MSA 아키텍처 설계, 서비스 디스커버리와 게이트웨이 구성
  • •분산 환경에서의 확장성과 유연성을 학습

Next.js를 활용한 서버사이드 렌더링(SSR)으로 SEO를 개선하고 초기 로딩 속도를 빠르게 만듭니다. MSA 아키텍처를 통해 큰 서비스를 작은 독립적인 서비스로 나누어 관리하는 방법을 배웁니다.

최신 풀스택 기술

  • •최신 JS/TypeScript 문법
  • •Serverless 아키텍처
  • •React-Node-Spring 혼합 구조 사례
  • •Micro Frontend 등 최신 트렌드

풀스택 개발의 최신 트렌드를 학습합니다. TypeScript는 Node.js 단계에서 선행 학습하며, Docker로 애플리케이션을 컨테이너화하여 배포 환경을 일관되게 관리합니다. Redis를 활용한 캐싱으로 성능을 개선하고, Serverless 아키텍처와 Micro Frontend 등 최신 기술을 실무 사례를 통해 이해합니다.

테스트 및 배포

  • •단위/통합 테스트(Jest, JUnit 등)
  • •CI/CD 파이프라인 구축
  • •자동화 배포 및 버전 관리
  • •무중단 배포와 성능 모니터링 기법

테스트 코드 작성은 안정적인 서비스를 만드는 필수 과정입니다. Jest와 JUnit으로 단위/통합 테스트를 작성하고, GitHub Actions로 CI/CD 파이프라인을 구축합니다. AWS의 EC2, S3, RDS 등 핵심 서비스를 활용하여 실제 클라우드 환경에 배포하며, 무중단 배포 전략으로 서비스 중단 없이 업데이트 하는 방법을 익힙니다. Java Spring 단계에서 함께 진행 됩니다.

2. 실전 프로젝트

기본 프로젝트

지금까지 학습한 React, Node.js, Spring을 하나로 통합한 풀스택 프로젝트를 만듭니다. 프론트엔드, 백엔드, 클라우드 세 분야를 모두 다루기 때문에 기본 프로젝트 기간의 오전 시간에도 진도를 계속 진행합니다. 동기/비동기 API를 구현하고, CI/CD로 자동 배포 환경을 구축하며, MSA 모듈을 연동하는 경험을 통해 기본적인 풀스택 개발 역량을 다집니다.

심화 프로젝트

Next.js의 SSR을 활용하여 더욱 복잡한 풀스택 서비스를 구축합니다. 생성형 AI 모델을 연동하여 실시간으로 콘텐츠를 생성하거나 사용자 질문에 응답하는 기능을 구현합니다. 대규모 트래픽을 견딜 수 있도록 서비스를 최적화하고, 보안을 강화합니다. 스크럼과 애자일 방법론을 실제 프로젝트에 적용하여 실무 프로세스를 경험하고, 산출물 문서화 훈련도 진행합니다.

실무 통합 프로젝트

부트캠프의 최종 단계로, 다른 과정(AI, 사이버 보안, 디자인 등) 수강생들과 협업하여 대규모 풀스택 프로젝트를 완성합니다. MSA와 Serverless를 혼합 적용하여 실무 수준의 아키텍처를 구현하고, 고가용성을 고려한 설계로 서비스가 중단되지 않도록 합니다. 실시간 모니터링 시스템을 구축하고, 장애 발생 시 대응 시나리오를 실습하며 최종 배포까지 완료합니다.

함께라서 가능한 스타트업 경험

TECH UP은 서로 다른 직군의 대규모 인원이 모여 하나의 팀으로 협업합니다. 기획, 디자인, 개발 등 전 직군이 스타트업 환경을 재현하며 실전을 경험합니다.

성장하는 스타트업의 팀 사이즈
